## ğŸ“š Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/satyasandeep007/DAGSnap
   cd DAGSnap
   ```

2. **Frontend**

   ```bash
   yarn install
   yarn dev
   ```

3. **Proxy Server**

   ```bash
   cd backend
   yarn install
   node server.js
   ```

4. **Snap**
   ```bash
   cd snap
   yarn install
   yarn start
   ```
